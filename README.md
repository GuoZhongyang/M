# 微博Mooc挖掘

## For What
通过对微博文本内容进行分析，使用NBSVM机器学习算法，发现微博上的 MOOCs 学习者。


## Prerequisites

The following needed:

* Python 2.7
* Numpy >= 1.12.0
* Scikit-learn >= 0.18.1
* pandas >= 0.19.2
* liblinear 2.11.2
* jieba >=0.38
* sklearn
* matplotlib >=2.0.2
* gensim >= 2.0.0
